# Svelte UI Elements

## Get started

### Installation

#### Using npm:

```bash
npm i svelte-ui-elements
```

#### Using yarn:

```bash
yarn add svelte-ui-elements
```

### Usage:

```js
<script>
  import { Button } from "svelte-ui-elements";
</script>

<main>
  <Button>test</Button>
</main>

```
